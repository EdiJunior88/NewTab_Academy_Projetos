Criar tabelas MySQL

Para começar, o comando de criação de tabela requer os seguintes detalhes –

  • Nome da tabela;
  • Nome dos campos;
  • Definições para cada campo.
  • Sintaxe
  • Aqui está uma sintaxe SQL genérica para criar uma tabela MySQL:
    CREATE TABLE table_name (column_name column_type);

Agora, vamos criar a seguinte tabela no banco de dados Newtab.

  create table newtab_tbl(
    id INT NOT NULL AUTO_INCREMENT,
    title VARCHAR(100) NOT NULL,
    author VARCHAR(40) NOT NULL,
    submission_date DATE,
    PRIMARY KEY ( id )
  );

Aqui, alguns itens precisam de explicação:

  • O atributo de campo NOT NULL está sendo usado porque não queremos que este campo seja NULL. Portanto, se um usuário tentar criar um registro com um valor NULL, o MySQL gerará um erro;
  • Atributo de campo AUTO_INCREMENT diz ao MySQL para ir em frente e adicionar o próximo número disponível ao campo id;
  • A palavra-chave PRIMARY KEY é usada para definir uma coluna como uma chave primária. Você pode usar várias colunas separadas por uma vírgula para definir uma chave primária.

Criação de tabelas a partir do prompt de comando

É fácil criar uma tabela MySQL a partir do prompt mysql>. Você usará o comando SQL CREATE TABLE para criar uma tabela.

Exemplo: Aqui está um exemplo, que criará newtab_tbl:

  root@host# mysql -u root -p

  Enter password:*******

  mysql> use Newtab;

  Database changed

  mysql> create table newtab_tbl(
    id INT NOT NULL AUTO_INCREMENT,
    title VARCHAR(100) NOT NULL,
    author VARCHAR(40) NOT NULL,
    submission_date DATE,
    PRIMARY KEY ( id )
  );
  
  Query OK, 0 rows affected (0.16 sec)

  mysql>

    OBSERVAÇÃO

    O MySQL não termina um comando até que você forneça um ponto-e-vírgula (;) no final do comando SQL.
