Subconsultas com a instrução DELETE

A subconsulta pode ser usada em conjunto com a instrução DELETE como com qualquer outra instrução mencionada acima.

A sintaxe básica é a seguinte.

    DELETE FROM TABLE_NAME
    
    [ WHERE OPERATOR [ VALUE ]

       (SELECT COLUMN_NAME

       FROM TABLE_NAME)

       [ WHERE) ]

Supondo que temos uma tabela CLIENTES_BKP disponível que é um backup da tabela CLIENTES. O exemplo a seguir exclui os registros da tabela CLIENTES para todos os clientes cuja IDADE seja maior ou igual a 27.

    SQL> DELETE FROM CLIENTES

       WHERE AGE IN (SELECT AGE FROM CLIENTES_BKP

          WHERE AGE >= 27 );

Isso afetaria duas linhas e, finalmente, a tabela CLIENTES teria os seguintes registros.
