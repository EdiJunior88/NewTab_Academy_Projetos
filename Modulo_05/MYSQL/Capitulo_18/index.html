STipos de dados

Definir adequadamente os campos em uma tabela é importante para a otimização geral do seu banco de dados. Você deve usar apenas o tipo e o tamanho do campo que realmente precisa usar. Por exemplo, não defina um campo de 10 caracteres de largura, se você sabe que usará apenas 2 caracteres. Esses tipos de campos (ou colunas) também são chamados de tipos de dados, de acordo com o tipo de dados que você armazenará nesses campos.

O MySQL usa muitos tipos de dados diferentes divididos em três categorias:

  • Numérico
  • Data e hora
  • Tipos de string.
