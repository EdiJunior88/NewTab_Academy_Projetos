Eliminar um banco de dados

Você precisará de privilégios especiais para criar ou excluir um banco de dados MySQL. Portanto, supondo que você tenha acesso ao usuário root, você pode criar qualquer banco de dados usando o binário mysql mysqladmin .

Tenha cuidado ao excluir qualquer banco de dados, pois você perderá todos os dados disponíveis nele.

Aqui está um exemplo para excluir um banco de dados (TUTORIAIS) criado no capítulo anterior:

    [root@host]# mysqladmin -u root -p drop Newtab
    Enter password:******

Isso lhe dará um aviso e confirmará se você realmente deseja excluir este banco de dados ou não.

Depois de se conectar ao servidor MySQL, é necessário selecionar um banco de dados com o qual trabalhar. Isso ocorre porque pode haver mais de um banco de dados disponível com o servidor MySQL.
