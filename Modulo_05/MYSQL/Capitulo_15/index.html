Criar um banco de dados

Você precisará de privilégios especiais para criar ou excluir um banco de dados MySQL. Então, supondo que você tenha acesso ao usuário root, você pode criar qualquer banco de dados usando o binário mysql mysqladmin.

Exemplo: aqui está um exemplo simples para criar um banco de dados chamado Newtab:

    [root@host]# mysqladmin -u root -p create Newtab
    Enter password:******

Isso criará um banco de dados MySQL chamado Newtab.
