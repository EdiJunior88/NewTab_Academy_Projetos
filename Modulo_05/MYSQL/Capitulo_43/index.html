Subconsultas com a instrução SELECT

As subconsultas também podem ser usadas com instruções INSERT. Dessa forma o INSERT utiliza os dados retornados da subconsulta para inserir na tabela.

A sintaxe básica é a seguinte.

    INSERT INTO table_name [ (column1 [, column2 ]) ]

      SELECT [ *|column1 [, column2 ]

      FROM table1 [, table2 ]

      [ WHERE VALUE OPERATOR ]

Considere uma tabela CLIENTES_BKP com estrutura semelhante à tabela CLIENTES. Agora, para copiar a tabela CLIENTES completa para a tabela CLIENTES_BKP, você pode usar a seguinte sintaxe.

    SQL> INSERT INTO CLIENTES_BKP

      SELECT * FROM CLIENTES

      WHERE ID IN (SELECT ID

      FROM CLIENTES) ;
