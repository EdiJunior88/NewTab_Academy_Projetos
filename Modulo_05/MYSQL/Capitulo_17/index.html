Seleção do banco de dados MySQL no prompt de comando

É muito simples selecionar um banco de dados no prompt mysql>. Você pode usar o comando SQL usado para selecionar um banco de dados.

Exemplo: aqui está um exemplo para selecionar um banco de dados chamado Newtab:

    [root@host]# mysql -u root -p
    Enter password:******
    mysql> use Newtab;
    Database changed
    mysql>

Agora, você selecionou o banco de dados Newtab e todas as operações subsequentes serão realizadas no banco de dados Newtab.

    OBSERVAÇÃO:

    Todos os nomes de bancos de dados, nomes de tabelas e nomes de campos de tabelas diferenciam maiúsculas de minúsculas.
    Portanto, você terá que usar os nomes próprios ao fornecer qualquer comando SQL.
