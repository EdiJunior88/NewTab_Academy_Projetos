Insert

Para inserir dados em uma tabela MySQL, você precisa usar o comando SQL INSERT INTO. Você pode inserir dados na tabela MySQL usando o prompt mysql> ou usando qualquer script como PHP.

Sintaxe

Aqui está uma sintaxe SQL genérica do comando INSERT INTO para inserir dados na tabela MySQL:

    INSERT INTO table_name ( field1, field2,...fieldN )
      VALUES
      ( value1, value2,...valueN );

Para inserir tipos de dados de string, é necessário manter todos os valores entre aspas duplas ou simples. Por exemplo, “valor” .

Inserindo dados do prompt de comando

Para inserir dados do prompt de comando, usaremos o comando SQL INSERT INTO para inserir dados na tabela newtab_tbl do MySQL.

O exemplo a seguir criará 3 registros na tabela newtab_tbl:

    root@host# mysql -u root -p password;

    Enter password:*******

    mysql> use Newtab;

    Database changed

    mysql> INSERT INTO newtab_tbl
      ->(title, author, submission_date)
      ->VALUES
      ->("Meu primeiro insert", "Fernando", NOW());

    Query OK, 1 row affected (0.01 sec)

    mysql> INSERT INTO newtab_tbl
      ->(title, title, submission_date)
      ->VALUES
      ->("Learn MySQL", "Anderson", NOW());

    Query OK, 1 row affected (0.01 sec)
  
    OBSERVAÇÃO

    Observe que todos os sinais de seta (->) não fazem parte do comando SQL. Eles estão indicando uma nova linha e são criados
    automaticamente pelo prompt do MySQL ao pressionar a tecla enter sem fornecer um ponto e vírgula no final de cada linha do comando.

No exemplo acima não fornecemos um id porque, no momento da criação da tabela, demos a opção AUTO_INCREMENT para este campo. Portanto, o MySQL se encarrega de inserir esses IDs automaticamente. Aqui, NOW () é uma função MySQL, que retorna a data e hora atuais.
