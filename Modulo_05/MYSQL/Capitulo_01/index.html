Porque aprender MySQL

MySQL é um banco de dados relacional que é utilizado para salvar dados em uma ou mais tabelas, nas quais os dados podem estar relacionados entre si. Utilizamos a linguagem para criar, modificar e extrair dados do banco de dados, bem como controlar o acesso do usuário.

MySQL atualmente é um dos bancos de dados mais populares com cerca de mais de 10 milhões de instalações pelo planeta. Quando navegamos por um site, inserimos, alteramos ou consultamos alguns dados de um sistema ou aplicativo, muitas vezes estamos utilizando informações que estão salvas nos bancos de dados como o MySQL.
