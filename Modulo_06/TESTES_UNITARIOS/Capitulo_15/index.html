Apresentação de React Testing Library

React Testing Library é um conjunto de utilitários que permitem testar componentes React sem depender dos detalhes de implementação.

Esses utilitários facilitam as consultas a DOM exatamente como um usuário faria, ou seja, é capaz de encontrar elementos de formulários, botões e links, entre outros, e por isso é uma ótima forma de testar elementos da UI.

  • Documentação oficial de React Testing Library: https://testing-library.com/docs/react-testing-library/intro/
