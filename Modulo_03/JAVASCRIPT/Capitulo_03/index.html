Características do JavaScript

JavaScript é uma das linguagens de programação mais populares do mundo. Uma ótima opção para sua primeira linguagem de programação.

Utilizamos principalmente JavaScript para criar:

  • Sites;
  • Aplicativos da web;
  • Aplicativos móveis usando ferramentas como React Native;
  • Programas para microcontroladores e a internet das coisas;
  • Aplicativos smartwatch…

E mais especificamente às suas características, considera-se que o JavaScript é:

Alto nível: fornece abstrações que permitem ignorar os detalhes da máquina em que está sendo executada. Ele gerencia a memória automaticamente com um coletor de lixo, para que você possa se concentrar no código em vez de gerenciar a memória como outras linguagens como C precisaria, e fornece muitas construções que permitem lidar com variáveis ​​e objetos altamente poderosos.

Dinâmico: ao contrário das linguagens de programação estáticas, uma linguagem dinâmica executa em tempo de execução muitas das coisas que uma linguagem estática faz no tempo de compilação. Isso tem prós e contras e fornece recursos poderosos, como digitação dinâmica, ligação tardia, reflexão, programação funcional, alteração do tempo de execução do objeto, fechamentos e muito mais. Não se preocupe se essas coisas forem desconhecidas – você vai entender um pouco mais em breve.

Tipos dinâmicos: uma variável não impõe um tipo. Você pode atribuir qualquer tipo a uma variável, por exemplo, atribuindo um número inteiro a uma variável que contém uma String.

Interpretado: é comumente conhecido como linguagem interpretada, o que significa que ela não precisa de um estágio de compilação para que um programa possa ser executado – em oposição a C, Java ou Go, por exemplo. Na prática, os navegadores compilam o JavaScript antes de executá-lo, por motivos de desempenho, mas isso é transparente para você – não há etapa adicional envolvida.

Multiparadigma : a linguagem não impõe nenhum paradigma de programação específico, ao contrário do Java, por exemplo, que força o uso da programação orientada a objetos por exemplo.

Ah! E caso você esteja se perguntando… Não, JavaScript não é o mesmo que Java, cada um é uma coisa, esta foi só uma má escolha de nome.
