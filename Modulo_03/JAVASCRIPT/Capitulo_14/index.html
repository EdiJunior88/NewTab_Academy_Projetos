Funções

Em qualquer programa JavaScript moderadamente complexo, tudo acontece dentro das funções. As funções são uma parte essencial e essencial do JavaScript.

Mas o que é uma função?

Uma função é um bloco de código independente.

Aqui está uma declaração de função:

    
      function getData() {
      // faz algo
      }

Uma função pode ser executada a qualquer momento, invocando-a, assim:

    
      getData();

Uma função pode ter um ou mais parâmetros:

    
      function getData() {
      // faz algo
      }
    
  
    
      function getData(color) {
      // faz algo
      }
    
  
    
      function getData(color, age) {
      // faz algo
      }

Quando podemos passar um parâmetro, chamamos a função que passa os parâmetros:

    
      function getData(color, age) {
      // faz algo
      }
      getData('green', 24);
      getData('black');

Observe que, na segunda chamada, passamos o parâmetro string black como argumento de color, mas sem age. Nesse caso, a age dentro da função é undefined.

Podemos verificar se um valor não é indefinido usando esta condicional:

    
      function getData(color, age) {
        if (typeof age !== 'undefined') {
        // faz algo
        }
      }

Você pode passar qualquer valor como parâmetro: números, sequências de caracteres, booleanos, arrays, objetos e também funções.

Uma função tem um valor de retorno. Por padrão, uma função retorna undefined, a menos que você adicione uma palavra-chave de return com um valor:

    
      function getData() {
        return 'olá mundo!';
      }

Podemos atribuir esse valor de retorno a uma variável quando chamamos a função:

    
      function getData() {
        return 'hi!';
      }
      let result = getData();

Você pode retornar apenas um valor.

Para retornar vários valores, você pode retornar um objeto ou um array, assim:

    
      function getData() {
        return ['Flavio', 37];
      }
      let [name, age] = getData();
