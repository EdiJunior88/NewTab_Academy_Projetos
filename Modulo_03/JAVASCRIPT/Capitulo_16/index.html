String

Uma string é uma sequência de caracteres. Também pode ser definido como uma string literal, entre aspas ou aspas duplas:

    
      'A string'
      “Outra string”

Pessoalmente, preferimos aspas simples o tempo todo e usamos aspas duplas apenas em HTML para definir atributos.

Você atribui um valor de string a uma variável como esta:

    
      const name = 'Flavio';

Você pode determinar o comprimento de uma string usando a propriedade length:

    
      'Bruno'.length; //5
      const name = 'Bruno';
      name.length; //5

Esta é uma string vazia: ” . Sua propriedade length é 0:

    
      ".length; //0

Duas strings podem ser unidas usando o operador + :

    
      "A " + "string";

Você pode usar o operador + para interpolar variáveis:

    
      const name = 'Junior';
      "Meu nome é " + name; //Meu nome é Junior

Os literais de modelo também são ótimos, pois fornecem uma maneira fácil de concatenar variáveis e expressões em seqüências de caracteres.

Você faz isso usando a sintaxe ${…} :

    
      const var = 'teste';
      const string = 'Isso é um ${var}'; //Isso é um teste

Dentro do ${} você pode adicionar qualquer coisa, até expressões:

    
      const string = Soma ${1 + 2 + 3}';
