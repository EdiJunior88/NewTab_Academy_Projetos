Variáveis e Constantes

Variáveis

Uma variável é uma posição nomeada de memória, que é usada para guardar um valor que pode ser modificado pelo programa. Ao desenvolvermos nossos algoritmos, frequentemente precisamos armazenar dados referentes ao problema, como um nome, um número ou mesmo o resultado de uma operação. Mas, para armazenar esses dados, precisamos solicitar ao computador que ele reserve uma área da memória para esse uso. A forma de solicitar ao computador que reserve memória é chamada de declaração de variáveis.

Por exemplo, se quisermos saber ao fim do algoritmo da lâmpada visto anteriormente quantas vezes a lâmpada foi trocada, basta modificá-lo para que ele tenha um contador:

    
      ...
      numero_de_vezes = 0
      ...
      Enquanto a lâmpada não acender
        numero_de_vezes = numero_de_vezes + 1
        ...

Desta forma, o numero_de_vezes é uma variável que é incrementada toda vez que a lâmpada não acender.

Qualquer tipo de informação simples pode ser guardada em uma variável. Alguns tipos mais comuns existentes em linguagens de programação são:

– Real: Declaramos variáveis do tipo numérico real para armazenar valores reais, em outras palavras, valores com ponto decimal (5.7, 3.2, -8.5). Esse seria o tipo ideal para armazenar, por exemplo, o salário de funcionários.

– Char (caractere): Declaramos variáveis do tipo literal caractere para armazenar um único caractere, que pode ser uma letra ou um símbolo. Por exemplo, para identificar o sexo do indivíduo, armazenaremos apenas o caractere ‘F’ ou ‘M’.

– String (cadeia): Declaramos variáveis do tipo literal cadeia para armazenar uma sequência de caracteres, ou seja, uma palavra, uma mensagem, um nome. Assim, se precisarmos de uma variável para armazenar o nome de uma pessoa, esse seria o tipo ideal.

– Boolean (lógico): Declaramos variáveis do tipo lógico para armazenar valores lógicos, ou seja, o valor de variáveis desse tipo será sempre VERDADEIRO ou FALSO.

– Array (vetor): Declaramos variáveis do tipo vetor quando queremos nos referir a um grupo de outras variáveis, de tipos semelhantes ou não. Identificamos cada uma das variáveis do conjunto através de um identificador sequencial (a posição da variável). Chamamos esse identificador de índice.

Constantes

Além de variáveis, que são conteúdos modificáveis pelo sistema, podemos estabelecer reservas de memória que não podem ser modificadas, que servem para guardar informações que se mantém iguais durante toda a execução do algoritmo.
