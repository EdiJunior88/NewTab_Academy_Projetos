Condicionais

📍 Para reforçar:

Com os operadores de comparação apresentados, podemos falar sobre condicionais. Uma instrução if é usada para fazer o programa seguir uma rota ou outra, dependendo do resultado de uma avaliação de expressão.

Este é o exemplo mais simples, que sempre se executa:

    
      if (true) {
      //Faça algo
      }

E pelo contrário, isso nunca é executado:

    
      if (false) {
      //nunca irá fazer algo
      }

O condicional verifica a expressão que você passa para um valor verdadeiro ou falso. Se você passa um número, ele sempre é avaliado como verdadeiro, a menos que seja 0. Se você passa uma sequência, ele sempre é avaliado como verdadeiro, a menos que seja uma sequência vazia. Essas são regras gerais de conversão de tipos para um booleano.

Você pode fornecer uma segunda parte para a instrução if: else.

Você anexa uma instrução que será executada se a condição if for falsa:

    
      if (true) {
      //faço algo aqui
      } else {
      //faça outra coisa aqui
      }

Como else aceita uma instrução, você pode aninhar outra instrução if / else dentro dela:

if (a === 10) { //faça algo no if } else if (b === 15) { //faça algo no else if } else { //senao faça algo aqui }
