Um pouco da sua história

Criado em 1995, o JavaScript percorreu um longo caminho desde seu começo. Foi a primeira linguagem de script que foi suportada nativamente pelos navegadores da Web e, graças a isso, ganhou uma vantagem competitiva sobre qualquer outra linguagem. Aliás, hoje ainda é a única linguagem de script que podemos usar para criar aplicativos da Web.

No começo, o JavaScript não era tão poderoso como é hoje, e era usado principalmente para animações sofisticadas. Ficou conhecido há anos atrás como HTML dinâmico.

Com as crescentes necessidades exigidas pela plataforma da Web (e que continua a exigir), o JavaScript também teve que crescer.

Agora o JavaScript também é amplamente usado fora do navegador. A ascensão do Node.js nos últimos anos desbloqueou o desenvolvimento de back-end, uma vez que o domínio de Java, Ruby, Python, PHP e linguagens mais tradicionais do lado do servidor.

Agora, o JavaScript também é o idioma que alimenta bancos de dados e muito mais aplicativos, e é ainda possível desenvolver aplicativos móveis, aplicativos de TV e muito mais. O que começou como um pequena linguagem dentro do navegador agora é a linguagem mais popular do mundo.

Às vezes, é difícil separar o JavaScript dos recursos do ambiente em que é usado. Por exemplo, a linha console.log() (que imprime os valores das variáveis no console dos navegadores) são um dos exemplos de código que não é JavaScript. Em vez disso, faz parte da vasta biblioteca de APIs fornecidas no navegador. Mas isso você também vai ver com o tempo…
