Porque aprender

Esse módulo nos fornece uma visão da linguagem JavaScript através de conceitos e fundamentos que irão nos ajudar no desenvolvimento de aplicações web.

JavaScript é uma linguagem muito poderosa, ela é muito mais que uma linguagem usada para pequenas validações de formulários, hoje é uma das principais e mais utilizadas por desenvolvedores ao redor do mundo através de uma comunidade gigantesca.

E nesse sentido, aprender JavaScript se torna mais fácil pelo simples fato de ter uma comunidade absurdamente grande para auxiliar. Imagine você não ter ideia de qual erro está acontecendo em seu código, no caso do JavaScript não precisa de alarde e muita preocupação. Basta uns minutos de pesquisa e seu problema será resolvido, tudo isso porque esta comunidade também é bastante unida e com disposição para ajudar. Aliás, este é um dos grandes motivos do JS (JavaScript) ser tão utilizado.
