O que aprender em JavaScript

  • Os tipos de variáveis;
  • Como trabalhar com os diferentes operadores;
  • Como trabalhar com Arrays;
  • As diferentes formas de fazer loops;
  • O que é programação assíncrona e callbacks;
  • Para que serve as promises;
  • Como fazer validação em JavaScript – e o que o HTML5 facilitou nesse assunto;
  • Como chamar uma API usando o XMLHttpRequest.
