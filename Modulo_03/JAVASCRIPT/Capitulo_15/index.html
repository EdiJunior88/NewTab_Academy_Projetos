Arrays

Um array é uma coleção de elementos. Arrays em JavaScript não são um tipo por si só, são objetos.

Podemos inicializar um array vazio dessas 2 maneiras diferentes:

    
      const a = [];
      const a = Array();

Você pode preencher previamente o array usando esta sintaxe:

    
      const a = [1, 2, 3];
      const a = Array.of(1, 2, 3);

Um array pode conter qualquer valor, mesmo valores de diferentes tipos:

    
      const a = [1, 'Flavio', ['a', 'b']];

Como podemos adicionar um array a um array, podemos criar arrays multidimensionais, que têm aplicações muito úteis:

    
      const arrayAux = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ];
      arrayAux[0][0]; //1
      arrayAux[2][0]; //7

Você pode acessar qualquer elemento do array referenciando seu índice, que começa do zero:

    
      arrayAux[0]; //1
      arrayAux[1]; //2
      arrayAux[2]; //3

Você pode inicializar um novo array com um conjunto de valores usando esta sintaxe, que primeiro inicializa um array de 12 elementos e preenche cada elemento com o número 0:

    
      Array(12).fill(0);

Você pode obter o número de elementos no array verificando sua propriedade length:

    
      const a = [1, 2, 3];
      a.length; //3

Como adicionar um item a um array

Podemos adicionar um elemento no final de um array usando o método push():

    
      a.push(4);

Podemos adicionar um elemento no início de uma array usando o método unshift():

    
      a.unshift(0);

Como remover um item de um array

Podemos remover um item do final de um array usando o método pop():

    
      a.pop();

Podemos remover um item do início de um array usando o método shift():

    
      a.shift();

Como juntar dois ou mais arrays

Você pode ingressar em vários arrays usando concat():

    
      const a = [1, 2];
      const b = [3, 4];
      const c = a.concat(b); //[1,2,3,4] a //[1,2] b //[3,4]

Outro método é o includes():

    
      a.includes(value);

Retorna true se a contiver value.

    
      a.includes(value, i);

Retorna true se a contiver value após a posição i.
