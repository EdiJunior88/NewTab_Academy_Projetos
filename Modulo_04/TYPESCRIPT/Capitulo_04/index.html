Funções

Escrever uma função em TypeScript é semelhante a escrevê-la em JavaScript, mas com parâmetros e tipo de retorno adicionados. Observe que qualquer função JavaScript é uma função TypeScript perfeitamente válida. No entanto, podemos fazer melhor adicionando tipo.

Sintaxe: vamos ver uma sintaxe básica de função TypeScript (com dois argumentos)

    function functionName(arg1: <arg1Type>, arg2: <arg2Type>): <returnType> {

      // Function body...
    }

Abaixo estão algumas funções para ajudar a entender melhor.

Exemplo 1: Neste exemplo, estamos escrevendo uma função para adicionar dois números

    // TypeScript Code
    // Following function returns the addition
    // of it's two parameters

    function addTwo(a: number, b: number): number {
     return a + b;
    }

    console.log(addTwo(7, 8)); // logs 15

Saída:

    15

Exemplo 2: adicione dois números e retorne uma string hexadecimal equivalente.

    // Following function adds it's two parameters then
    // returns it as string form in base 16
    
    function getHexAddition(a: number, b: number): string {
      return (a + b).toString(16);
    }

    console.log(getHexAddition(10, 16)); // logs '1a'

Saída:

    1a

Adicionando Parâmetros Opcionais e Padrão: Adicionar um parâmetro opcional é super simples, basta adicionar? ao final do nome do argumento.

Exemplo 3: Escreva uma função que registre uma mensagem de bom dia com um nome, se um nome não for passado, registre apenas o bom dia.

    // Following function returns good morning
    // message based on value of name passed

    function goodMorning(name?: string): string {
      // if name exits use it as suffix else ignore name

      const suffix = (name ? `, ${name}.` : '.');
      return 'Good Morning' + suffix;
    }

    // logs 'Good Morning.'
    console.log(goodMorning());

    // logs 'Good Morning, Sam.'
    console.log(goodMorning('Sam'));

Saída: Para argumento padrão, o sufixo com um sinal de igual e valor padrão (o compilador TS deduzirá automaticamente o tipo do argumento padrão com base no valor fornecido).

    Good Morning.
    Good Morning, Sam.

Exemplo 4: Escreva uma função que retorna a base ^ {potência}, se a potência não for fornecida, será considerado 1.

    function pow(base: number, power = 1): number {
      return Math.pow(base, power);
    }

    console.log(pow(7)); // logs 7
    console.log(pow(7, 2)); // logs 49

Saída:

    7
    49
