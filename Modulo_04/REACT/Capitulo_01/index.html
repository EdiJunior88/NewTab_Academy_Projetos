Porque aprender React

Nos últimos anos a evolução do acesso à internet impulsionou uma grande onda de adesão ao desenvolvimento web e, consequentemente, ao Javascript (essa é, aliás, a razão de estarmos nessa capacitação, certo?). Esse crescimento quer dizer que no passado recente se desenvolveu um riquíssimo ecossistema em torno da linguagem, na forma de plugins, extensões, bibliotecas e frameworks – que às vezes cresceram ao ponto de gerarem ecossistemas próprios. Existem diversas alternativas no mercado de desenvolvimento de software, mas definitivamente a mais popular delas é o React.

Como possivelmente o framework Javascript mais popular da atualidade, é quase que auto-explicativo o porquê de o usarmos: abre portas a muitas oportunidades de trabalho, possui uma comunidade extensa e ativa (muito útil para resolver problemas), conta com uma ampla variedade de extensões e plugins que facilitam o trabalho no dia a dia, excelente suporte dos desenvolvedores (que é o próprio Facebook, por sinal), etc.

Outra vantagem é que o React foi projetado com encapsulamento e componentização em mente, o que quer dizer que é bem simples dividir nosso projeto em pedaços mais fáceis de manter. Além disso, mais do que seus competidores principais (Vue e Angular) o React depende do Javascript para realizar suas operações, o que pode ser uma vantagem se você se sentir mais confortável usando a linguagem.
