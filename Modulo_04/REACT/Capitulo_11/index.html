Componentes e Containers

Até aqui vimos como a sintaxe especial do React funciona, suas particularidades e como realizar algumas rotinas, como listar elementos dinamicamente ou exibi-los condicionalmente. Agora podemos passar para o coração e a alma do React: componentes. Eles são o fundamento desta biblioteca e podemos fazer coisas incríveis com eles. Basicamente componentes são blocos menores de código que representam a lógica e layout de uma fração da página que podem ser reutilizados, mas há certas diferenças entre os do React e do Angular (por exemplo) importantes que precisamos manter em mente. Os vídeos vão mostrar um pouco mais sobre isso.
