Usando componentes e propriedades

Dentro da sua pasta src crie um diretório para armazenar seus componentes. Pode nomeá-la como quiser, desde que fique claro o que essa pasta contém, ok? Vamos chamá-la aqui de components mesmo.

Dentro dela vamos criar um componente simples para uma lista de compras. Vamos chamá-lo de Lista.js. E sim, é com letra maiúscula mesmo. Lembra que dissemos que o JSX reconhece código com letra minúscula como HTML e maiúscula como componente/JS? Então, para mantermos o padrão de nomeação e assim manter o código organizado e livre de possíveis confusões vamos nomear nosso componente em letra maiúscula desde o início. Esse é um padrão comum e praticamente todo código React será assim, então vai se acostumando.

Dentro dele vamos criar um componente funcional mesmo:

    import React from 'react';
    
    const lista = (props) => {
      return (
        <div>
          <ul>
            <li>Alface</li>
            <li>Tomate</li>
            <li>Queijo</li>
            <li>Macarrão</li>
            <li>Arroz</li>
          </ul>
        </div>
      )
    };
    
    export default lista;

Mas espera, porque aqui dentro nós não estamos usando letras maiúsculas? Muito simples: porque aqui dentro não é preciso! Não estamos importando o próprio componente, então não quebramos nenhum padrão deste modo. Pelo contrário, como estamos lidando com uma variável e não uma classe é mais "padrão" nomearmos iniciando com letra maiúscula. O que importa é importarmos com letra maiúscula, e o default que colocamos no final do componente vai garantir que o componente será importado corretamente.

Agora vamos voltar para o arquivo App.js que o nosso instalador criou. Junto com as demais importações, logo no começo do arquivo, vamos importar nosso novo componente:

    import Lista from './components/Lista';

A partir de agora, podemos integrar nossa listinha de compras juntamente com a nossa saída em JSX. Se dentro do return do App.js colocarmos ele assim:

    return (
        <div className="App">
          <header className="App-header">
            <img src={logo} className="App-logo" alt="logo" />
            <p>
              Edit <code>src/App.js</code> and save to reload.
            </p>
    
            <a
              className="App-link"
              href="https://reactjs.org"
              target="_blank"
              rel="noopener noreferrer"
            >
              Learn React
            </a>
    
            <Lista />
          </header>
        </div>
    );

Desta forma, com apenas uma linha de código conseguimos colocar uma lista na tela. Algumas observações antes de fecharmos essa parte: no import omitimos a extensão do .js porque o importador, por padrão, interpreta que arquivos sem extensão são tratados como Javascript. Também repare que, aqui usamos letra maiúscula na chamada. É aqui que mais importa essa nomeação: como somos obrigados a chamar componentes usando letra maiúscula de todo modo, aqui é que precisamos declarar o componente como tal. Em nosso arquivo original essa preocupação não existe porque graças à palavra chave default o componente é sempre retornado corretamente, independente de como o chamamos mais tarde.
