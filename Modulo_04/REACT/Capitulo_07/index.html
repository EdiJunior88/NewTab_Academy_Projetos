Componentes

Conforme abordamos brevemente na introdução, componentes são os “bloquinhos” que podemos criar dentro do React.

No caso do React não temos o conceito de templating fora do componente. Em vez disso, todo o código do template é definido dentro do próprio arquivo.

Essa abordagem pode inicialmente parecer mais desorganizada, mas permite maior granularidade de componentes sem tanta estrutura de código o que, no fim das contas, acaba sendo tão ou mais organizado.

Além disso, como a criação de componentes no React não é dependente de módulos, dependemos exclusivamente da interface de import e export do JavaScript.
