JSON

JSON é uma formatação de dados estruturados baseada em texto. A ferramenta é muito leve e serve para troca de informações entre aplicações por meio de APIs.

Podemos codificar um array em string JSON utilizando a função json_encode

Podemos decodificar uma string JSON para um objeto utilizando json_decode

Constante

É uma referência para um identificador de valor único. Isso significa que, durante a execução, uma constante não poderá receber qualquer outro valor após ser definida.
