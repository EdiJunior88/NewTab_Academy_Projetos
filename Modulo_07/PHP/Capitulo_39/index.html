Respostas

  1. Qual a diferença de declarar string com aspas duplas ou aspas simples?

    2. Aspas duplas interpretam valores de variáveis, assim evitamos concatenação. Aspas simples entrega apenas a string.

  2. Qual a finalidade do método nativo print_r?

    3. Imprime uma variável de forma legível. Isso inclui array e objetos.

  3. Qual método podemos utilizar para apresentar o valor e tipo de determinada variável?

    4. var_dump()

  4. Quais variáveis globais utilizamos em formulários e quando usar cada uma?

    5. $_GET: os dados do formulário serão enviados pela url formando uma query string.

    $_POST: os dados do formulário serão enviados no corpo da requisição e não pela url.

  5. O que é o Composer?

    6. O Composer é um gerenciador de dependências em PHP. Com ele é possível instalar e atualizar bibliotecas, deixando-as prontas para uso dentro do projeto.
