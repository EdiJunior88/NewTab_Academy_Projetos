Por que aprender? Para que serve e quem o utiliza?

Após estruturarmos nossa página web com o HTML, estilizar com o CSS e adicionar interatividade e dinamismo com o Javascript, chegou a hora de aplicarmos as ações, validações e funcionalidades da nossa aplicação, e é aí que entra o desenvolvimento backend. Um exemplo prático: ao fazer login no site da Newtab, os dados digitados são enviados pro servidor (requisição) para processamento e procura no banco de dados. Caso os dados enviados correspondam com algum registro armazenado, o servidor devolve a resposta de sucesso, sua sessão é iniciada e você pode realizar seus estudos. A responsabilidade dessa tarefa é de uma linguagem server-side e aprenderemos a desenvolver com uma das mais utilizadas no mercado: PHP.

Segundo o Stackshare, mais de 40.000 empresas utilizam o PHP como ferramenta, incluindo Facebook, Slack e WhatsApp, entre outras. A boa aceitação da linguagem para o desenvolvimento backend deve-se a algumas vantagens que o PHP nos traz, como:

  • Facilidade no aprendizado devido a sintaxe simples;

De acordo com o site w3techs, pelo menos até o início de 2021, estima-se que quase 80% dos sites na internet utilizam PHP, um número bastante relevante visto a quantidade de sites que existem atualmente. Esse dado reforça a ideia de alta demanda por profissionais qualificados no mercado de trabalho.
