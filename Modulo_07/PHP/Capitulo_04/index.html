Opção 2 – Pacote de instalação

O objetivo dessa instalação é facilitar a preparação do ambiente já que o pacote possui a ferramenta que utilizaremos na formação.

O XAMPP é um software livre, multiplataforma para servir soluções web e facilitar o desenvolvimento local. Seu acrônimo consiste em:

X – Multiplataforma;

A – Apache (servidor);

M – MariaDB (banco de dados);

P – Perl (linguagem);

P – PHP (linguagem);

Passo a passo para instalação no Windows: https://pt.wikihow.com/Instalar-o-XAMPP-para-Windows

Passo a passo para a instalação no Linux: https://pt.wikihow.com/Instalar-o-XAMPP-no-Linux
