Porque aprender HTML

Imagine que você é um eletricista contratado para fazer uma instalação elétrica em uma casa em construção e, ao chegar no local, descobre que os pedreiros sequer ergueram a primeira parede. Impossível trabalhar assim, não é? Da mesma forma, é praticamente impossível criar aplicações, rotinas e serviços web sem usar o HTML.

Como o alicerce e a alvenaria dão a estrutura à casa necessária para que outras camadas sejam colocadas (como piso, teto, rede elétrica, encanamento e pintura), o HTML dá a estrutura a uma página web sobre a qual são escritas as rotinas, interações com o usuário, estilização e funcionalidades necessárias. Mesmo sem precisar trabalhar diretamente com o HTML (o que é raro), é inevitável interagir com ele no desenvolvimento front-end, quer seja criando estilos via CSS, funções Javascript ou até mesmo arquivos de back-end (em PHP ou Python, por exemplo).

Assim, é importante conhecer suas regras, estrutura e principais componentes se quiser progredir nessa área. Mas não se preocupe: o HTML é bem intuitivo em seu funcionamento, e antes que você perceba, estará fazendo todo o sentido!
