Atributos e elementos

Atributos

Agora você pode estar pensando: “legal aprender sobre tags e tudo… mas minha página é apenas uma tela branca chata!”. E com toda razão, no momento não temos nenhum conteúdo para adicionar. Antes de passarmos para como adicionarmos e manipularmos conteúdo na página, vamos explicar algumas coisas importantes que afetam a página como um todo pelos bastidores.

Já vimos como usamos tags no HTML para definir elementos da página, certo? Por enquanto vimos elementos tão gerais que só precisamos declará-los uma vez. Mas pense em uma página normal: quantas imagens, parágrafos e links não aparecem? Como diferenciamos entre eles, ou atribuímos características diferentes para certos elementos, mas não para outros? Por meio de dos chamados atributos.

Pense neles como dados pessoais: você possui características individuais, como CPF, RG, personalidade, etc. e outras que você compartilha com outras pessoas, como primeiro nome, data de nascimento ou altura. Da mesma forma, os atributos definem características dos elementos HTML, podendo ajudar a descrevê-lo ou identificá-lo.

Nós os declaramos dentro das tags, e com isso queremos dizer dentro mesmo, antes de fechar a primeira metade da tag, e seguindo o formato:

<nome do atributo>="<valor do atributo>".

Vamos dar um exemplo simples: se temos uma tag de parágrafo (vamos vê-la com mais detalhes mais pra frente) e queremos identificá-la, podemos usar o atributo id, que vai associar aquele elemento a um texto único (isto é, não podemos repeti-lo em outro elemento) de identificação, assim:

<p id="primeiro">
  Lorem ipsum, dolor sit amet...
</p>
<p id="segundo">
  Lorem ipsum, dolor sit amet...
</p>

No momento, isso não nos é tão útil assim, mas mais para frente veremos que esse atributo é um dos mais cruciais quando manipulamos a estrutura/estética do HTML com Javascript e CSS. Cada elemento possui um conjunto de atributos, alguns universais, outros únicos, e na exposição dos elementos abaixo veremos mais atributos. Mas o que importa para nós neste momento é que você entenda a “fórmula” para declarar os atributos e o seu conceito, para não se perder. Se você tiver dúvidas sobre os atributos de uma tag específica, ou os mais globais que não forem abordados nesta capacitação, basta pesquisá-los, é bem fácil de achá-los.
