Style

<style></style>

Acabamos de comentar que, para fins de organização, é legal usar o link para adicionar conteúdo .css. Mas… sabemos que nem tudo são flores e às vezes o melhor é colocar os estilos na própria página. O que fazer, então? Usar a tag <style>. Esse elemento abre uma espécie de “janela” no nosso documento HTML, permitindo tratar tudo dentro dela como se fosse um arquivo CSS. Não vamos entrar muito no mérito do CSS em si porque vamos vê-lo com mais detalhes mais pra frente, mas é bom sabermos dessa estrutura desde agora.

Como é um elemento que vai encapsular um conteúdo (no caso, código) precisamos abrir e fechar, assim:

<head>
 <title>Exemplo</title>
 <style type="text/css">
  /*Todo código aqui dentro é CSS!*/
  body {
   background: red;
  }
 </style>
</head>
