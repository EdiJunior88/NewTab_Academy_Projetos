Animação simples

Com CSS é possível animar elementos sem o uso de Javascript. Uma animação é a transição de um estado para outro, no CSS funciona assim: tenho um elemento, com tais propriedades. Ao passar o mouse (por exemplo, já que animações podem ocorrer sem o usuário interagir com um elemento) sobre ele, quero que mude de propriedades gradualmente. Na prática, um elemento de fundo vermelho ficaria gradualmente com fundo azul no :hover, e não mudaria de repente para o azul, sem transição nenhuma, por exemplo.

No CSS, podemos fazer animações simples (de um estado para outro, “dois passos”) bem como animações mais complexas (vários “passos” dentro de uma mesma animação).

Para uma animação simples, podemos usar apenas transições entre dois estados, que já chegaremos ao resultado esperado.

A seguir você encontra uma animação em funcionamento, bem como seu código: https://codepen.io/anna-erbetta-furlan/pen/YzXxxYK. As transições sempre são colocadas no estado inicial (ou seja, no caso do exemplo, o estado sem o hover).

Um bom lugar para fazer transições, animações e transformações é nesse gerador: https://www.css3maker.com/css3-transition.html. Note que além de propriedades como transition, há propriedades como -webkit-transition, -moz-transition… Elas fazem o mesmo, porém cada uma é “adaptada” para diferentes browsers.

Dentro de uma transição, você pode notar mais de um valor, exemplo:

transition: all 1s 16 ease;

all: indica a propriedade que será atingida por essa transição. No caso, “all” indica que todas as propriedades que mudam de um elemento para outro serão afetadas por essa transição. Se tivéssemos um elemento em que a cor do texto e a cor do fundo mudam, mas só quiséssemos que a mudança da cor do fundo fosse afetada pela transição, escreveríamos: transition: background 1s ease;

1s: indica o tempo, em segundos, que a transição irá durar.

ease: o modo como essa transição se dará. Você pode encontrar todos os modos e seus exemplos aqui. Também é possível “brincar” com essas curvas e gerar uma transição personalizada (aqui podemos ter acesso a um gerador).
