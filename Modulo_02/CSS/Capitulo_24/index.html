Bootstrap

Criado em 2011 por Jacob Thorton e Mark Otto, o Bootstrap é um framework front-end que coloca padrões e elementos HTML, CSS e JS para facilitar o desenvolvimento de páginas web. Foi feito como uma tentativa de resolver incompatibilidades de desenvolvimento entre os membros do Twitter da época.

Basicamente, esta ferramenta conta com diversos componentes prontos para unificar e facilitar a maneira como sites são construídos e leva também a responsividade.

Na prática, a principal aplicação do Bootstrap é na criação de sites responsivos. Esta facilidade se deve ao fato de que ele possui vários plugins em JavaScript (jQuery) que tornam o seu dia a dia muito mais fácil. A ferramenta ajuda o profissional a implementar recursos como o menu dropdown, carousel, modal, slideshow, etc., que são aplicados com muito mais facilidade. De acordo com o site oficial, o Bootstrap “usa CSS tradicional, mas seu código fonte utiliza os dois pré-processadores CSS mais populares, Less e Sass”.

Esse framework é bem simples de ser implementado e pode ser usado tanto via CDN (Content Delivery Network, neste caso, os arquivos ficam no servidor do Bootstrap e são carregados no site) como também localmente fazendo o download dos arquivos. A biblioteca usa um arquivo .css e alguns scripts que definem os componentes disponíveis para uso.
