Porque aprender CSS

O CSS permite mudar estilos de elementos do site todo de uma vez – que por HTML seriam mudados um a um. Por exemplo: se existem dois blocos de textos separados, ambos com a fonte em um tamanho de 12px, ao invés de mudarmos para 16px nos dois elementos, poderíamos selecioná-los no CSS e mudá-los de uma vez em uma folha de estilo. Agora imagine esta situação em um site de 20 páginas! Bem mais fácil mudar um arquivo só, certo?

Além da praticidade e aumento de possibilidades na hora de customizar um site, o CSS possui extrema importância em sua parte visual, pois além de deixá-lo atrativo, de modo mais fácil, garante uma experiência do usuário mais rica. Permite o uso de cores para indicações, hierarquia visual de informações, interações de feedback… e outras características que proporcionam uma melhor visualização e interação do usuário com uma interface.

E não é exatamente isso que um profissional de front-end mais quer? É isto que precisamos!
