Float e clear

Existem também duas propriedades que influenciam no posicionamento de um elemento na página:

float: elementos com float “flutuam” na página, ou seja, deixam de ocupar o espaço que ocupavam antes e passam a ficar em cima (ou embaixo, dependendo do z-index) dos elementos à sua volta. Possui os seguintes valores:

  • float: none; o elemento não flutua.
  • float: left; o elemento flutua à direita do container que estiver.
  • float: right; o elemento flutua à esquerda do container que estiver.

clear: especifica em quais lados de um elemento elementos flutuantes não irão flutuar. Qual lado ficará “limpo” de outros elementos. Possui os seguintes valores:

  • clear: none; os elementos flutuantes continuam à volta do elemento com clear:none.
  • clear: left; a esquerda do elemento será “limpa” de elementos flutuantes, que não serão “permitidos”.
  • clear: right; elementos não serão permitidos “flutuarem” à direita do elemento com essa propriedade.
  • clear: both; ambos os lados (direita e esquerda) ficarão sem outros elementos.

Aqui podemos ver um exemplo: https://www.w3schools.com/cssref/tryit.asp?filename=trycss_class-clear. Remova a “class=”clear”” da tag <p> para ver o efeito.
