Backgrounds

Background é uma propriedade que aplica fundos a elementos (cor sólida, imagens, gradientes). As propriedades que modificam o fundo de um elemento são:

background-color: define a cor sólida de um fundo (ex: background-color: #000;).

background-image: define uma imagem como fundo. É usada uma URL para definir qual imagem será (ex.: background-image: url(‘caminho/da/imagem.jpg’);).

background-repeat: define se o background, no caso de uma imagem que não cobre o todo do elemento em que está aplicada, irá repetir (background-repeat: repeat;) ou não (background-repeat: no-repeat;), bem como o modo como irá repetir (só verticalmente (background-repeat: repeat-y; ou só horizontalmente background-repeat: repeat-x;).

background-attachment: se o fundo será fixo ou não (ex.: background-attachment: fixed; faz o background continuar onde está enquanto a página rola).

background-position: a posição do fundo pode ser dada em pixels, porcentagem e em qualquer outra unidade de medida.

background-size: o tamanho da imagem do background. Pode ser dado em unidades de medida ou em valores como cover (cobrir), contain (conter), etc.
